The Office VR



The Office VR is a Virtual Reality game created within Unity3D using C# and Object-Oriented Programming. The Office VR has been managed by GitScrum, a project management platform specific to software development with an emphasis on agile concepts. As a developer, this has been an interesting mix of abstract problem solving, dynamic programming and harnessing many of Unity3D’s features to solve a complex problem.

The Berry Exchange

A dynamic berry auctioning site created from scratch using laravel5. Languages used are PHP, HTML, CSS and SQL for the database. Users can register to start their own auctions. other users can bid on auctions. personal information is stored in a SQL database and this can be changed on the users account page.

A Tiered Bookstore Application

Developed in .Net with C# and SQL, the purpose of this application is to connect many different applications through to the same database to simulate a modern and real online ordering system through an API whilst considering concurrent connections and software design patterns.

This online ordering system is composed of an API, SQL database, an Administrator and a client application.
